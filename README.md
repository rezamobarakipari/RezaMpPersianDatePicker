# RezaMpPersianDatePicker 🇮🇷

A lightweight, customizable Persian (Shamsi / Jalali) DatePicker component for **Blazor**  
with **Theme**, **Size**, and **Popup / Inline** modes.

---

## ✨ Features

- 📅 Persian (Jalali) calendar based on `PersianCalendar`
- 🎨 Multiple themes (Metro, Dark, Minimal)
- 📐 Multiple sizes (Small, Medium, Large)
- 🪟 Popup mode & Inline (side) mode
- ⚡ Fully customizable via CSS variables
- 🧩 Blazor-friendly API
- 📦 NuGet-ready structure
- 🧼 No external UI dependencies

---

## 🚀 Demo

A demo page is included in the solution to preview:
- Theme switching
- Size switching
- Popup / Inline mode
- Two-way binding with selected date
